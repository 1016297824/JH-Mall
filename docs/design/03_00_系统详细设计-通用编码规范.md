## 零、通用编码规范

> 所有编码模块（有 Mapper 层）的强制规范，详见各模块详细设计 §2.1.1。

### Mapper 层规则

| 操作 | 用什么 | 说明 |
| --- | --- | --- |
| **SELECT** | `LambdaQueryWrapper` + `BaseMapper.selectPage()` | 类型安全，字段变更 IDE 即时感知 |
| **UPDATE** | `@Update` 注解 | 简洁直读，算术运算 `SET field = field + #{value}` 必须用 |
| **INSERT** | `BaseMapper.insert()` | MyBatis-Plus 内置 |
| **DELETE** | `BaseMapper.update()` 软删除 | 统一逻辑删除 `set is_deleted = 1` |

### Lombok 规则

| 注解 | 适用场景 |
| --- | --- |
| `@Data` | DTO / VO / Request / Response / DO |
| `@Slf4j` + `@RequiredArgsConstructor` | Service / Controller / Consumer |

### 接口与实现命名

- 接口文件以 `I` 前缀命名：`IUserProfileService`、`IAddressService`
- 实现类在 `service/impl/` 下，类名后缀 `Impl`：`UserProfileServiceImpl`
- 控制器不重复模块名：`ProfileController` 而非 `UserProfileController`
