## 二、错误码详细定义

遵循阿里巴巴 Java 开发手册（嵩山版）错误码规约。

### 2.1 通用规则

- 成功码：`"00000"`
- 错误码格式：`来源字母 + 四位数字编号`，共 5 位，字符串类型
- 来源含义：**A**=用户端错误，**B**=系统执行出错，**C**=调用第三方服务出错
- 所有错误码通过追加方式扩展，已确定的编号永久固定

### 2.2 返回结构

C 端使用独立响应体，不依赖若依管理端 `AjaxResult`。

**Java 类：** `MallResult<T>`（`com.mall.api.dto`），定义见 `05_mall-api契约层设计.md` §7。

```json
{
  "errorCode": "00000",
  "errorMessage": "操作成功",
  "userTip": "",
  "data": {},
  "requestId": "..."
}
```
