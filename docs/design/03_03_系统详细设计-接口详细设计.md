## 三、接口详细设计

> 本章列出各个服务对外提供的接口，按服务划分小节。每个接口包含路径、方法、请求参数、响应参数、错误码。
> 网关层路由规则见概要设计 8.1 路由前缀。

### 3.1 mall-auth 认证接口

mall-auth 是 C 端用户专属认证中心（9301 端口），独立于管理端 `ruoyi-auth`（9201）。Token 体系使用 JWT，与管理端完全隔离（不同 issuer、secret、Redis key 前缀），支持多端共存（手机 + PC + Pad 可同时在线，每端独立 token）。

**通用约束：**

- 协议：生产环境使用 HTTPS
- 响应体统一结构：

  ```json
  {
    "errorCode": "00000",
    "errorMessage": "操作成功",
    "userTip": "",
    "data": {},
    "requestId": "..."
  }
  ```
- HTTP 状态码：200（成功）、400（参数错误）、401（未登录）、403（无权限）、429（频率限制）、500（系统错误）
- JSON key 统一使用 lowerCamelCase
- 时间格式统一为 `"yyyy-MM-dd HH:mm:ss"`
- 超大整数（如用户 ID）以 String 类型返回
- 错误码遵循阿里规范 A/B/C 格式，详见第二章错误码表

**接口清单：**

| #  | 方法   | 路径                               | 说明                                   | 需登录 | 审计日志 |
| -- | ------ | ---------------------------------- | -------------------------------------- | :----: | :------: |
| 1  | POST   | `/api/auth/users`                | 手机号注册（验证码+密码+隐私协议同意） |   否   |          |
| 2  | POST   | `/api/auth/sms_codes`            | 发送短信验证码                         |   否   |          |
| 3  | POST   | `/api/auth/sessions`             | 手机号+密码登录（创建会话）            |   否   |    ✓    |
| 4  | POST   | `/api/auth/sessions/sms`         | 短信验证码登录                         |   否   |    ✓    |
| 5  | POST   | `/api/auth/sessions/refresh`     | 刷新 access_token                      |   否   |          |
| 6  | DELETE | `/api/auth/sessions/current`     | 退出登录（删除当前会话）               |   是   |    ✓    |
| 7  | POST   | `/api/auth/wechat/sessions`      | 微信授权登录（小程序 code 换 token）   |   否   |    ✓    |
| 8  | POST   | `/api/auth/wechat/phone_binding` | 微信用户绑定手机号                     |   是   |    ✓    |
| 9  | PUT    | `/api/auth/phone`                | 换绑手机号                             |   是   |    ✓    |
| 10 | PUT    | `/api/auth/password`             | 通过验证码重置密码                     |   否   |          |
| 11 | PUT    | `/api/auth/password`             | 通过旧密码修改密码                     |   是   |    ✓    |
| 12 | DELETE | `/api/auth/account`              | 注销账号                               |   是   |    ✓    |
| 13 | GET    | `/api/auth/sessions/current`     | 校验 token 有效性                      |   是   |          |

**错误码：** 详见第二章错误码表，auth 域涉及 A01xx（注册）、A02xx（登录）、A03xx（权限）、C0220（短信服务）、C0230（微信服务）

#### 3.1.1 POST /api/auth/users — 手机号注册

**功能说明：** 新用户通过手机号+验证码注册，同时设置密码并同意隐私协议。注册成功后自动登录，返回 token。

**请求体：**

| 字段            | 类型    | 必填 | 说明                                 |
| --------------- | ------- | :--: | ------------------------------------ |
| phone           | String  |  是  | 手机号                               |
| password        | String  |  是  | 密码，长度 8~32 位，需包含字母和数字 |
| smsCode         | String  |  是  | 短信验证码，6 位数字                 |
| isPrivacyAgreed | Integer |  是  | 是否同意隐私协议，必须为 1           |

**成功响应（errorCode=00000）：**

```json
{
  "errorCode": "00000",
  "errorMessage": "注册成功",
  "userTip": "",
  "data": {
    "userId": "1723984712345678",
    "accessToken": "eyJhbGciOiJIUzI1NiJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiJ9...",
    "tokenType": "Bearer",
    "expiresIn": 1800
  },
  "requestId": "a1b2c3d4-..."
}
```

| 返回字段     | 类型    | 说明                            |
| ------------ | ------- | ------------------------------- |
| userId       | String  | 用户 ID，超大整数以 String 返回 |
| accessToken  | String  | 访问令牌，有效期 30 分钟        |
| refreshToken | String  | 刷新令牌，有效期 7 天           |
| tokenType    | String  | 固定值 `Bearer`               |
| expiresIn    | Integer | 过期秒数，1800 秒 = 30 分钟     |

**错误码：**

| errorCode | 触发条件                               |
| --------- | -------------------------------------- |
| A0101     | 未同意隐私协议（isPrivacyAgreed != 1） |
| A0151     | 手机号已被注册                         |
| A0152     | 手机号格式校验失败                     |
| A0121     | 密码长度不够或不符合复杂度要求         |
| A0131     | 短信验证码错误                         |
| A0132     | 短信验证码已过期                       |
| A0401     | 必填参数为空                           |

#### 3.1.2 POST /api/auth/sms_codes — 发送短信验证码

**功能说明：** 向指定手机号发送 6 位短信验证码。需防刷（同一手机号 60 秒内不可重复发送，同一 IP 每日上限）。

**请求体：**

| 字段  | 类型   | 必填 | 说明                                                                                        |
| ----- | ------ | :--: | ------------------------------------------------------------------------------------------- |
| phone | String |  是  | 手机号                                                                                      |
| scene | String |  是  | 使用场景：`register` / `login` / `reset_password` / `change_phone` / `bind_phone` |

**成功响应（errorCode=00000）：**

```json
{
  "errorCode": "00000",
  "errorMessage": "验证码已发送",
  "userTip": "",
  "data": {
    "expireIn": 300
  },
  "requestId": "..."
}
```

| 返回字段 | 类型    | 说明                            |
| -------- | ------- | ------------------------------- |
| expireIn | Integer | 验证码有效秒数，300 秒 = 5 分钟 |

**错误码：**

| errorCode | 触发条件                                                   |
| --------- | ---------------------------------------------------------- |
| A0152     | 手机号格式校验失败                                         |
| A0410     | 请求频率过高（同一手机号 60 秒内重复发送，或同一 IP 超限） |
| A0401     | 必填参数为空                                               |

### 3.7 mall-search 接口

mall-search 覆盖商品**搜索引擎**，统一为 9307 端口。C 端路由 `/api/search/**`（可匿名或 C 端 token），管理端由 discovery.locator 自动路由（`/mall-search/{controllerPath}`）。通用约束同 3.1。

**C 端接口清单（可匿名）：**

| # | 方法 | 路径                         | 说明                 |
| - | ---- | ---------------------------- | -------------------- |
| 1 | GET  | `/api/search`              | 商品全文搜索         |
| 2 | GET  | `/api/search/suggestions`  | 搜索建议（自动补全） |
| 3 | GET  | `/api/search/hot_keywords` | 热门搜索词           |

**管理端接口清单：**

| # | 方法 | 路径                              | 说明                          | 权限码                        |
| - | ---- | --------------------------------- | ----------------------------- | ----------------------------- |
| 4 | POST | `/mall-search/index/operations` | 索引操作（全量重建/增量同步） | `mall:search:index:rebuild` |
| 5 | GET  | `/mall-search/index/status`     | 索引状态查询                  | `mall:search:index:sync`    |

**错误码：** 详见第二章错误码表，search 域涉及 A08xx（搜索处理异常）

---

#### 3.7.1 GET /api/search — 商品全文搜索

**功能说明：** 基于 ES 的商品全文搜索。可匿名访问。

**查询参数：**

| 参数       | 类型    | 必填 | 说明                                                          |
| ---------- | ------- | :--: | ------------------------------------------------------------- |
| keyword    | String  |  是  | 搜索关键词                                                    |
| page       | Integer |  否  | 默认 1                                                        |
| size       | Integer |  否  | 默认 20，最大 100                                             |
| categoryId | Integer |  否  | 类目筛选                                                      |
| brandId    | Integer |  否  | 品牌筛选                                                      |
| minPrice   | Integer |  否  | 最低价（分）                                                  |
| maxPrice   | Integer |  否  | 最高价（分）                                                  |
| sort       | String  |  否  | `default` / `price_asc` / `price_desc` / `sales_desc` |

**成功响应：** 分页结构，`records` 每条含 `spuId`、`name`、`mainImage`、`minPrice`、`salesCount`、`rating`。

**错误码：** A0801（搜索参数输入有误）

---

#### 3.7.2 GET /api/search/suggestions — 搜索建议

**查询参数：**

| 参数    | 类型    | 必填 | 说明              |
| ------- | ------- | :--: | ----------------- |
| keyword | String  |  是  | 输入前缀          |
| size    | Integer |  否  | 返回条数，默认 10 |

**成功响应：** `{ "suggestions": ["iPhone", "iPad", "AirPods"] }`。

**错误码：** A0401（keyword 为空）

#### 3.7.3 GET /api/search/hot_keywords — 热门搜索词
