# 若依（RuoYi-Cloud）系统总览与架构

## 一、简介

RuoYi-Cloud 是基于 Spring Cloud 微服务架构的企业级快速开发平台，提供 RBAC 权限管理、代码生成、定时任务、文件服务、监控等功能模块。本系统基于 v3.6.8 二次开发。

## 二、系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    客户端（Vue 3 + Element Plus）            │
│    ruoyi-ui（管理端）         mall-client（商城 C 端）        │
└──────────────────────┬──────────────────────────────────────┘
                       │ HTTP/REST
┌──────────────────────▼──────────────────────────────────────┐
│              API 网关（ruoyi-gateway :8080）                 │
│    路由转发 · 统一鉴权 · 限流熔断 · XSS 防护 · 验证码          │
└──┬─────────┬──────────┬──────────┬──────────┬───────────────┘
   │         │          │          │          │
┌──▼──┐ ┌───▼───┐ ┌───▼───┐ ┌───▼───┐ ┌───▼──────────────┐
│认证 │ │系统    │ │代码   │ │定时   │ │  文件服务          │
│授权 │ │管理    │ │生成   │ │任务   │ │ (ruoyi-file)      │
│auth │ │system │ │gen    │ │job    │ │                  │

└──┬──┘ └───┬───┘ └───┬───┘ └───┬───┘ └──────────────────┘

          │  SpringBoot Admin  │
          │  (ruoyi-visual     │
          │   :9100)           │
          └───────────────────┘
```

## 三、微服务模块一览

| 服务          | 端口 | 说明                         |
| ------------- | ---- | ---------------------------- |
| ruoyi-gateway | 8080 | Spring Cloud Gateway 网关    |
| ruoyi-auth    | 9200 | 认证授权中心                 |
| ruoyi-system  | 9201 | 系统管理模块                 |
| ruoyi-gen     | 9202 | 代码生成模块                 |
| ruoyi-job     | 9203 | 定时任务模块                 |
| ruoyi-file    | 9300 | 文件服务模块                 |
| ruoyi-monitor | 9100 | 监控中心（SpringBoot Admin） |

## 四、公共模块（ruoyi-common）

| 子模块                  | 说明                                                      |
| ----------------------- | --------------------------------------------------------- |
| ruoyi-common-core       | 核心模块：常量、工具类、异常体系、统一响应 R、分页、Excel |
| ruoyi-common-security   | 安全模块：注解鉴权、Feign 拦截、Token 管理、全局异常      |
| ruoyi-common-redis      | Redis 缓存封装                                            |
| ruoyi-common-log        | 操作日志注解 + AOP 切面                                   |
| ruoyi-common-datasource | 多数据源注解（@Master / @Slave）                          |
| ruoyi-common-datascope  | 数据权限注解 + AOP 切面                                   |
| ruoyi-common-sensitive  | 数据脱敏（手机/邮箱/身份证等）                            |
| ruoyi-common-swagger    | SpringDoc API 文档自动配置                                |

## 五、外部依赖

| 组件          | 用途                         |
| ------------- | ---------------------------- |
| Nacos         | 服务注册 + 配置中心（:8848） |
| Sentinel      | 网关流控 + 熔断降级（:8718） |
| Redis         | 缓存、登录状态、字典缓存     |
| MySQL         | 业务数据库                   |
| MinIO（可选） | 文件存储                     |

## 六、数据流（以一次请求为例）

```
浏览器 → ruoyi-gateway (8080)
  ├─ AuthFilter 鉴权（JWT + Redis 校验）
  ├─ ValidateCodeFilter 验证码校验（登录接口）
  ├─ XssFilter XSS 过滤
  └─ BlackListUrlFilter IP 黑名单
       ↓
  ruoyi-auth (9200) — 登录认证，签发 Token
       ↓
  ruoyi-system (9201) — 业务处理（携带 Token 在 Header 中）
       ↓
  返回统一格式 R/AjaxResult → 前端解析渲染
```
