# 系统工具功能

系统工具包含代码生成、表单构建和系统接口文档。

## 1. 代码生成

### 功能作用
根据数据库表结构自动生成 Controller、Service、Mapper、实体类、前端 Vue 页面等全套 CRUD 代码。

### 页面路径
`/tool/gen` — `views/tool/gen/index.vue`

### 权限标识
| 操作 | 权限标识 |
|------|----------|
| 查询 | `tool:gen:list` |
| 新增 | `tool:gen:code` |
| 修改 | `tool:gen:edit` |
| 删除 | `tool:gen:remove` |

### 后端 API
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/gen/list` | 代码生成列表 |
| GET | `/gen/db/list` | 查询数据库表列表 |
| POST | `/gen/importTable` | 导入表结构 |
| GET | `/gen/preview/{tableId}` | 预览代码 |
| GET | `/gen/download/{tableName}` | 下载代码（zip） |
| GET | `/gen/genCode/{tableName}` | 生成到自定义路径 |
| GET | `/gen/batchGenCode` | 批量生成 |
| GET | `/gen/synchDb/{tableName}` | 同步数据库表 |

### 使用流程

**步骤 1：导入表**
- 点击"导入"，选择数据库中要生成代码的表
- 后端读取表结构和字段信息，写入 `gen_table` 和 `gen_table_column`

**步骤 2：配置生成信息**
- **基本信息**：包名、模块名、业务名、功能名、作者
- **字段配置**：列表显示、查询方式（= / != / like / between）、表单类型（输入框/下拉框/日期选择器等）、是否必填
- **生成信息**：模板类型（单表/树表）、生成路径、上级菜单

**步骤 3：预览/下载**
- 预览所有生成文件的代码
- 下载 zip 包或直接生成到项目目录

### 支持的文件生成类型
```
Controller       — Controller 层
Service          — Service 接口 + 实现
Mapper           — Mapper 接口
Mapper.xml       — MyBatis XML 映射
Domain           — 实体类
Domain.java      — 实体类（含 Swagger 注解）
index.vue        — 前端列表页
data.vue         — 前端编辑弹窗
sql.zip          — 菜单 SQL 脚本
```

### 模板引擎
基于 Apache Velocity 模板引擎，模板文件位于 `ruoyi-gen` 模块的 `resources/vm/` 目录下。

### 代码生成配置
可在 `application.yml` 中配置：
- 作者名
- 包名
- 表前缀（自动去除）
- 是否覆盖文件

## 2. 表单构建

### 功能作用
可视化拖拽表单设计器，生成前端代码。

### 页面路径
`/tool/build` — `views/tool/build/index.vue`

### 实现技术
纯前端实现，基于 Vue + 拖拽组件，不在后端生成存储。

### 支持的表单组件
| 组件类型 | 说明 |
|----------|------|
| 输入框 | 文本输入 |
| 数字输入框 | 数字类型 |
| 选择器 | 下拉选择 |
| 单选/多选 | Radio / Checkbox |
| 日期/时间 | DatePicker / TimePicker |
| 开关/滑块 | Switch / Slider |
| 文本域/富文本 | Textarea / Editor |
| 上传/颜色 | Upload / ColorPicker |
| 按钮/分割线 | Button / Divider |

### 使用流程
1. 左侧组件列表拖拽到中间画布
2. 右侧属性面板配置字段名、标签、校验规则等
3. 点击"导出"生成 Vue 表单代码

## 3. 系统接口

### 功能作用
基于 SpringDoc（OpenAPI 3.0）生成在线 API 文档。

### 页面路径
`/tool/swagger` — 内嵌 Swagger UI 页面

### 实现方式
- `ruoyi-common-swagger` 模块自动配置 SpringDoc
- 网关模块 `SpringDocConfig` 聚合各服务的 API 文档
- 各 Controller 使用 Swagger 注解（`@Tag` / `@Operation` / `@Schema`）描述接口

### 访问地址
```
http://localhost:8080/swagger-ui/index.html
```
