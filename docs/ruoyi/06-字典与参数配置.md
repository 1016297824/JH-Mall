# 字典与参数配置

## 1. 字典管理

### 功能作用
管理系统数据字典，提供统一的数据字典查询和缓存机制。

### 页面路径
`/system/dict` — `views/system/dict/index.vue`

### 后端 API

#### 字典类型
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/dict/type/list` | 字典类型列表 |
| POST | `/dict/type` | 新增字典类型 |
| PUT | `/dict/type` | 修改字典类型 |
| DELETE | `/dict/type/{dictIds}` | 删除字典类型 |
| DELETE | `/dict/type/refreshCache` | 刷新字典缓存 |

#### 字典数据
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/dict/data/list` | 字典数据列表 |
| GET | `/dict/data/type/{dictType}` | 根据类型获取数据 |
| POST | `/dict/data` | 新增字典数据 |
| PUT | `/dict/data` | 修改字典数据 |
| DELETE | `/dict/data/{dictCodes}` | 删除字典数据 |

### 缓存机制
- 字典数据全量缓存到 Redis，Key 格式：`sys_dict:{dictType}`
- 首次访问时从数据库加载，写入 Redis
- 字典数据变更时自动刷新缓存
- 前端通过 Vuex `dict` 模块管理字典数据

### 字典类型示例
| 字典类型 | 说明 | 数据示例 |
|----------|------|----------|
| `sys_normal_disable` | 正常/停用 | 0=正常, 1=停用 |
| `sys_user_sex` | 用户性别 | 0=男, 1=女, 2=未知 |
| `sys_yes_no` | 是否 | Y=是, N=否 |
| `sys_notice_status` | 公告状态 | 0=草稿, 1=发布 |

### 前端使用方式
```vue
<el-select v-model="form.status">
  <el-option
    v-for="dict in dict.type.sys_normal_disable"
    :key="dict.value"
    :label="dict.label"
    :value="dict.value"
  />
</el-select>
```

### 实现流程
1. 前端组件设置 `dicts: ['sys_normal_disable']` 声明依赖
2. `dict` 全局混入自动加载字典数据（从 Vuex 或 API 获取）
3. 字典数据缓存在 `dict` Vuex 模块中
4. 字典变更后调用 `DELETE /dict/type/refreshCache` 清除 Redis 缓存

## 2. 参数配置

### 功能作用
管理系统配置参数，支持根据配置键名查询。

### 页面路径
`/system/config` — `views/system/config/index.vue`

### 后端 API
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/config/list` | 参数配置列表 |
| POST | `/config/export` | 导出参数配置 |
| GET | `/config/{configId}` | 参数详情 |
| GET | `/config/configKey/{configKey}` | 根据键名查询 |
| POST | `/config` | 新增参数 |
| PUT | `/config` | 修改参数 |
| DELETE | `/config/{configIds}` | 删除参数 |

### 常用内置配置
| 配置键 | 说明 |
|--------|------|
| `sys.user.initPassword` | 初始密码 |
| `sys.account.registerUser` | 是否开放注册 |
| `sys.account.chrtype` | 密码字符类型规则 |
| `sys.account.initPasswordModify` | 初始密码是否强制修改 |
| `sys.account.passwordValidateDays` | 密码过期天数 |

### 使用方式
```java
String value = configService.selectConfigByKey("sys.account.registerUser");
// 或
@Value("${sys.account.registerUser}")
private String registerUser;
```

### 缓存机制
- 参数配置缓存到 Redis，Key 格式：`sys_config:{configKey}`
- 新增/修改/删除后自动刷新缓存
