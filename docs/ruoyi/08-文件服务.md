# 文件服务

## 功能作用
提供文件上传、下载、删除的统一服务，支持本地存储和 MinIO 对象存储。

## 服务信息
- 端口：9300
- 模块：`ruoyi-file`

## 后端 API
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/upload` | 文件上传 |
| DELETE | `/delete` | 文件删除 |

## 存储方式

### 1. 本地存储（默认）
- 文件保存在 `{profile}/uploadPath/` 目录
- 通过静态资源映射 `/profile/**` 访问
- 配置：`application.yml` 中的 `ruoyi.profile`

### 2. MinIO 对象存储
- `MinioConfig` 配置 MinIO 客户端
- 支持分布式文件存储
- 配置：`application.yml` 中的 `minio` 配置段
  - `endpoint`：MinIO 服务地址
  - `accessKey` / `secretKey`：访问密钥
  - `bucketName`：存储桶名称

## 文件上传实现
```java
public interface ISysFileService {
    String uploadFile(SysFile sysFile) throws Exception;
    Boolean deleteFile(SysFile sysFile) throws Exception;
}

// 本地实现
public class LocalSysFileServiceImpl implements ISysFileService { ... }

// MinIO 实现
public class MinioSysFileServiceImpl implements ISysFileService { ... }
```

### Referer 防盗链
- `RefererFilter` 过滤非本站引用
- 防止文件被外部站点直接引用

### 文件上传限制
- 文件名长度限制
- 文件大小限制（默认 50MB）
- 扩展名白名单校验
- 支持图片压缩/缩略图

## 前端上传组件
直接使用 Element UI 的 `el-upload` 组件：
```html
<el-upload
  :action="uploadUrl"
  :headers="headers"
  :on-success="handleUploadSuccess"
  :before-upload="beforeUpload">
  <el-button size="small" type="primary">点击上传</el-button>
</el-upload>
```
