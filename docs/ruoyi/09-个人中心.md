# 个人中心

## 功能作用
用户管理个人信息，包括基本资料修改、密码重置、头像上传。

## 页面路径
`/user/profile` — `views/system/user/profile/index.vue`

## 后端 API
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/user/profile` | 获取个人信息 |
| PUT | `/user/profile` | 修改个人信息 |
| PUT | `/user/profile/updatePwd` | 修改密码 |
| POST | `/user/profile/avatar` | 上传头像 |

## 功能模块

### 1. 基本信息
- 显示：用户名、昵称、手机号、邮箱、性别、所属部门
- 支持修改：昵称、手机号、邮箱、性别

### 2. 修改密码
- 需要输入：旧密码、新密码、确认密码
- 密码强度校验（长度、字符类型）
- 校验旧密码是否正确
- BCrypt 加密存储

### 3. 头像上传
- 上传到文件服务（本地/MinIO）
- 支持裁剪/预览
- 更新到用户信息

## 实现流程

### 修改个人资料
1. `GET /user/profile` 获取当前用户信息
2. 前端表单预填
3. `PUT /user/profile` 提交修改
4. 后端校验唯一性（手机号/邮箱）
5. 更新 `sys_user` 表

### 修改密码
1. 输入旧密码和新密码
2. `PUT /user/profile/updatePwd` 提交
3. 后端校验旧密码是否正确
4. 新密码 BCrypt 加密后更新
5. 清除 Redis 缓存中密码错误计数

### 密码策略提醒
- 初始密码未修改 → 弹窗提示修改密码
- 密码超过有效期 → 弹窗提示修改密码
- 有效期由参数配置 `sys.account.passwordValidateDays` 控制
