## 一、设计一致性检查（对照 00_MASTER.md）

### 1.1 色彩系统

| 元素 | 设计规范 | 实际实现 | 状态 |
|------|---------|---------|:--:|
| 主色 Primary | `#7C3AED` | `tokens.css --el-color-primary: #7C3AED` | ✅ |
| CTA 行动色 | `#22C55E` | 定义了 `$color-cta` 但**未用于任何按钮**，登录按钮用的是 `type="primary"`（紫色） | ⚠️ |
| 中性色系列 | 50~900 阶 | `variables.scss` 和 `tokens.css` 完整定义 | ✅ |
| 文本颜色 | Primary `#4C1D95` / Secondary `#A855F7` | 正确引用 CSS Variables | ✅ |
| 价格色 | 设计系统未定义 | `#EF4444`（红）硬编码在 ProductCard 中 | ⚠️ |

### 1.3 间距与栅格

## 三、可访问性（A11y）检查

### 3.1 WCAG 2.1 AA 关键项

| # | 检查项 | 发现 | 严重度 |
|:--|------|------|:--:|
| 1 | 文本对比度 ≥ 4.5:1 | `#4C1D95` 标题 / `#FAF5FF` 背景 ≈ **9.5:1** ✅；`#A855F7` 辅助 / `#FAF5FF` ≈ **3.2:1** ❌ | 高 |
| 2 | 焦点可见 `:focus-visible` | **全局缺失** — 无任何 `:focus-visible` 样式 | 严重 |
| 3 | 键盘操作 | Banner `@click` **无** `@keydown`；购物车按钮无键盘支持 | 高 |
| 4 | 图片 alt | ProductCard `:alt="product.spuName"` ✅；Banner `:alt="banner.title"` ✅；CategoryGrid `:alt="cat.name"` ✅ | OK |
| 5 | 表单标签 | 搜索框有 `placeholder` 但**无** `<label>` 关联 | 中 |
| 6 | 语义化 HTML | `<main>` / `<section>` / `<h2>` / `<h3>` 正确使用 | OK |
| 7 | ARIA 标注 | 购物车按钮**无** `aria-label`；Logo 链接**无** `aria-label` | 中 |
| 8 | 错误播报 | `ElMessage.error` **无** `role="alert"` — 屏幕阅读器不会播报 | 高 |
| 9 | `prefers-reduced-motion` | ✅ 完整实现 | OK |
| 10 | 颜色作为唯一指示 | 价格用红色 `#EF4444` — 红绿色盲可能无法区分 | 低 |

### 3.2 ui-ux-pro-max 通用规则

| 规则 | 发现 |
|------|------|
| 禁用 Emoji 图标 | ProductSection 标题 "🔥 热销推荐" — Emoji 不是功能性图标，属于标题文本装饰，**可接受** |
| Hover 不改变布局 | CategoryGrid `scale(1.04)` — 轻微放大但 Grid 内不偏移 ✅ |
| 点击元素有 `cursor:pointer` | ProductCard ✅、Banner ✅、CategoryGrid（隐式通过 router-link）✅ |

## 四、改进建议

### P0 — 必须修复

| # | 问题 | 位置 | 修复方案 |
|:--|------|------|---------|
| 1 | **无焦点可见样式** | 全局 | 在 `element-overrides.css` 添加 `:focus-visible { outline: 2px solid var(--el-color-primary); outline-offset: 2px; }` |
| 2 | **Banner 点击无键盘支持** | `BannerSwiper.vue:30` | 给 `.banner-slide` 添加 `@keydown.enter="handleBannerClick(banner)"` 和 `tabindex="0"` |
| 3 | **辅助文字对比度不足** | `tokens.css` | `--el-text-color-secondary` 从 `#A855F7` 改为 `#7E22CE`（达到 4.5:1） |
| 4 | **接口失败未保留缓存** | `HomePage.vue:23-24` | catch 块不要清空 `categories`/`hotProducts`，只 toast 提示 |

### P1 — 建议修复

| # | 问题 | 位置 | 修复方案 |
|:--|------|------|---------|
| 5 | **动效缩放值与设计规范不一致** | `ProductCard.vue:69` `scale(1.05)`；`CategoryGrid.vue:64` `scale(1.04)` | 统一改为 `scale(1.02)` |
| 6 | **类目空态未隐藏区域** | `HomePage.vue:38` | CategoryGrid 加 `v-if="categories.length > 0"` |
| 7 | **错误消息无 aria-live** | `HomePage.vue:24` | `ElMessage.error` 无法加 aria，改用页面内 `<div role="alert">` 或忽略（Element Plus 限制） |
| 8 | **购物车按钮无 aria-label** | `BaseNavBar.vue:34` | 添加 `aria-label="购物车"` |
| 9 | **硬编码值替换为变量** | `ProductSection.vue:66/89/106` 的 `12px/48px/16px` | 替换为 `$spacing-sm/$spacing-2xl/$spacing-md` |

### P2 — 可选优化

| # | 问题 | 位置 | 修复方案 |
|:--|------|------|---------|
| 10 | CTA 色 `$color-cta` 定义了但未使用 | `variables.scss:4` | 后续"加入购物车"按钮使用 `$color-cta` |
| 11 | 搜索框无 `<label>` | `BaseNavBar.vue:22` | 添加 `aria-label="搜索商品"`（等同于 label） |
| 12 | 价格红色对红绿色盲不友好 | `ProductCard.vue:93/100` | 价格色从 `#EF4444` 改为 `#DC2626`（更深红）并配合 `¥` 符号作为双重指示 |
| 13 | Logo 链接无 aria-label | `BaseNavBar.vue:18` | 添加 `aria-label="JH-Mall 首页"` |

## 五、总结

| 维度 | 评分 | 说明 |
|:--|:--:|------|
| 设计一致性 | **7/10** | 色彩/字体/间距基本对齐，动效缩放值有偏差，硬编码需替换为变量 |
| 交互完整性 | **8/10** | 核心流程完整，类目空态和错误恢复需补 |
| 可访问性 | **4/10** | 焦点可见、键盘操作、对比度、ARIA 多方面缺失 |
| **综合** | **6/10** | 功能完整但无障碍需重点补强 |

**4 个 P0 问题必须在上线前修复**，重点是焦点可见样式和 Banner 键盘操作。P1 建议在下一轮迭代中修复。
