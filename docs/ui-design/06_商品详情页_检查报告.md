## 三、交互流程检查（对照 15_前端详细设计-C端.md §2.3）

|------|:--:|------|

| 库存展示 | ✅ | `watch(currentSku)` 拉 `getSkuDetail` 获取 `availableStock` |
| 购物车 / 购买 | ✅ | `postCartItem` + `buyNow` |

| 问题 | 严重度 | 说明 |
|------|:------:|------|

## 五、改进建议汇总

### 🔴 P0 — 无（全部已修复）

05 报告的 7 个 P0 全部解决。

### 🟡 P1 — 强烈建议（4 项）

| # | 问题 | 修复方案 |
|---|------|---------|
| 1 | SKU 详情加载中无反馈 | `watch` 中设置 `skuLoading` 标记，展示骨架或加载文字 |
| 2 | 划线价 `#a0b4c4` 对比度不足 | 改为 `#7a94a6`（≥ 3:1 大文本） |
| 3 | 立即购买未校验登录 | `buyNow()` 前检查 `localStorage.accessToken`，无则跳 `/login?redirect=...` |
| 4 | SKU 选择器缺 ARIA | 在 `SkuSelector.vue` 中为选中按钮添加 `aria-checked`，父容器 `role="radiogroup"` |

### 🟢 P2 — 可选优化（4 项）

| # | 问题 | 修复方案 |
|---|------|---------|
| 5 | 缩略图按钮缺 `aria-label` | 添加 `:aria-label="'切换到第' + (item.idx + 1) + '张图'"` |
| 6 | 库存变化无播报 | `.stock-info` 添加 `aria-live="polite"` |
| 7 | 服务承诺 `::before` 不可访问 | 用 `<span aria-hidden="true">✓</span>` 替代 CSS `content` |
| 8 | 按钮 active scale 不一致 | 统一为 `0.97`（规范值） |
